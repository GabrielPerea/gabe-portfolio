# Gabe Perea - Portfolio Site

Personal portfolio site built with React + Vite. Deployed on Vercel.

## Local development

```bash
npm install
npm run dev
```

## Deploy

Push to GitHub, connect repo to Vercel. Auto-deploys on every push.

## Editing content

All content lives in `src/Portfolio.jsx`. Update the data arrays at the top of the file to change case studies, tools, career history, or copy.
